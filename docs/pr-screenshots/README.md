# PR screenshots

UI workflow screenshots are committed under branch-specific folders in this directory.

- Path format: `docs/pr-screenshots/<branch-name>/`
- Required baseline captures:
  - `register-open.png`
  - `admin-pre-start.png`
  - `admin-post-start.png`
  - `public-post-start.png`
- Add extra screenshots when a UI change needs more focused visual evidence.
